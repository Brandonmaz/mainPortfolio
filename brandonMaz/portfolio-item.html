Planets ofStar Wars

JS/Ajax/CSS

Stepping into our first project with General Assembly, I wanted to do something different.

I came into this project not knowing what I would be able to accomplish and set my expectations somewhere in the middle. I shocked myself a bit becuase I was able to create a carousel without using bootstrap and that carousel was able to move cycle through the pictures with each planet name popping up. So if I were to access the star wars API and change the planets I was looking for, the names would change automatically. I was also able to create a modal that pops up the information of one planet. I was unable to get description of all the planets on my list. Which is where I fell a little short. This is the only project that doesn't have a live link and I decided to deploy it to Heroku, but ran into some problems
