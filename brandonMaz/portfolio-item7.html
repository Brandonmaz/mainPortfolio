MERN Stack

This is the second group project with a UX/UI team. There were four developers on this application, along with three UX/UI members.

Since I had already created a todo app with a group already, I wanted to make this one a little bit different by adding more individuality to it. We started off thinking we could create a simple todo app, then convert it to a notification app. With only a week to accomplish this task, we were not able to make that specific goal, but came up with a wonderful todo app. We decided to go with Authorization on this one to make each users list is specific to them. I worked with two other developers on the backend creating the apps functionality. We each broke off from there and took different parts of the front end design. My contribution was the reminders page, which has all the todo items. I enjoyed working closely with the UX/UI team and would love to do more projects with a group like that.
